# Veni Vici - Discover App

## Overview
This is a React web application that allows users to discover random dog breeds and learn more about them. Users can click a "Discover" button to display a random dog and view its details. Users can also click attributes to add or remove them from a ban list.

## Features
- [x] Displays a title and description of the app
- [x] Shows one item at a time
- [x] Displays an image for each item
- [x] Displays at least three attributes (breed, origin, group)
- [x] Clicking "Discover" shows a random new item
- [x] Attributes are clickable
- [x] Clicking attributes adds/removes them from a ban list
- [x] Ban list is displayed on screen
- [x] Ban list items can be removed by clicking them

## How It Works
- Click **Discover** to generate a random dog
- Click on breed, origin, or group to add it to the ban list
- Click items in the ban list to remove them
- Ban list prevents unwanted items from appearing

## Tech Stack
- React
- Vite
- JavaScript
